# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you find a security vulnerability, please bring it to our attention immediately.

### How to Report

Please do NOT report security vulnerabilities via public GitHub issues.

Instead, please report them by emailing the project maintainer. You should receive a response within 48 hours.

### Response Process

1.  **Triage**: We will confirm the issue and determine its severity.
2.  **Fix**: We will work on a patch to resolve the vulnerability.
3.  **Release**: We will release a security update and notify users.

Thank you for improving the security of this project.
